#### 6.3 Physics로 움직임 제어하기

- Physics는 Rigidbody 컴포넌트와 Collider 컴포넌트로 구성
    1. Rigidbody 컴포넌트 : 힘 계산(물체에 작용하는 중력이나 마찰 등) 담당
    2. Collider 컴포넌트 : 물체의 충돌 판정
- Physics는 무대 위를 플레이어가 자유롭게 이동하는 액션 게임이나 복잡한 충돌 판정이 필요한 슈팅 게임에 이상

#### 6.4 콜라이더 모양 조정하기

<img width="196" alt="스크린샷 2021-07-10 오후 2 56 59" src="https://user-images.githubusercontent.com/43170505/125153501-15f6c980-e18f-11eb-9f13-f53e0bb530a2.png">

플레이어가 구름 위에 제대로 올라서 있다.

    여러가지 Find 메서드
    - Find(오브젝트이름) : 씬 중 오브젝트 이름과 일치하는 게임 오브젝트를 한 개 찾아서 반환
    - FindWithTag(태그이름) : 씬 중 태그 이름과 일치하는 게임 오브젝트를 한 개 찾아서 반환
    - FindGameObjectsWithTag(태그이름) : 씬 중 태그 이름과 일치하는 게임 오브젝트를 여러 개 찾아서 반환. 반환값은 GameObject 배열이다.
    - FindObjectOfType(타입이름) : 씬 중 타입 이름과 일치하는 게임 오브젝트를 한 개 찾아서 반환
    - FindObjectsOfType(타입이름) : 씬 중 태그 이름과 일치하는 게임 오브젝트를 한 개 찾아서 반환. 반환값은 Object 배열이다.

#### 6.9 Physics를 사용해 충돌 판정하기

- Physic를 이용한 충돌 판정
    1. Collision 모드 (충돌 모드) : 오브젝트가 닿았을 때 충돌 판정은 물론 충돌했을 때 주변에 미치는 영향까지 포함해 충돌 반응을 실행
    2. Trigger 모드 (통과 모드) : 충돌 판정만 하고 충돌 응답은 실행하지 않음 (충돌한 오브젝트가 빠져나감)
